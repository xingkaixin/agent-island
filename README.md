# AgentIsland

macOS 菜单栏应用，实时监控 AI 编程 Agent 的运行状态。

## 功能概览

**支持监控的 Agent**
- Claude Code
- Codex
- Cursor

**菜单栏状态指示**
- 图标动态反映当前状态：空闲、运行中（脉冲动画）、需要关注
- 标题栏显示紧凑计数，如 `2·1·3`（运行·空闲·待处理）
- 悬浮提示显示详细状态

**会话面板**
- 点击菜单栏图标弹出浮窗，查看所有活跃会话
- 每个会话显示：Agent 类型、状态标签（thinking/tool/shell/mcp/file 等）、工作目录、已用时间
- 需要用户关注时顶部显示醒目横幅

**Hook 注入管理**
- 一键注入/移除/恢复 Agent Hook 配置
- 自动备份原始配置，支持一键回滚
- 显示各 Agent 的注入状态和最近活动

**日志中心**
- 统一时间线查看所有 Agent 事件
- 按来源、事件类型、Session ID、时间范围过滤
- 展开查看原始 JSON 数据

**系统通知**
- 当 Agent 发起权限请求或需要用户回到终端时，发送系统通知
- 可在设置中开关通知

**开机自启动**
- 支持设置开机自动启动（macOS LaunchAgent）
